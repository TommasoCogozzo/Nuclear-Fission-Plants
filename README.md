# Nuclear Fission Plants - Politecnico di Torino

This repository contains the technical reports and MATLAB simulation scripts developed for the **Nuclear Fission Plants** course at Politecnico di Torino. 

The project works focus on the thermal-hydraulic, thermo-mechanical, and fluid dynamics analysis of nuclear reactor cores, with particular emphasis on Pressurized Water Reactors (PWRs) and two-phase flow phenomena.
